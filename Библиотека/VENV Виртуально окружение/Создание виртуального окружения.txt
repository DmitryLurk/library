Необходимо ввести команду в каталоге приложения, 

python -m venv venv

для смены папок можно использовать команду 
cd .. переход вверх по каталогу
cd folder переход в указанную folder

Что бы активировать виртуальное окружение необходимо переуйти в 

project_folder\venv\Scripts

и выполнить скрипт activate.bat

что бы выйти из окружения необходимо прописать в терминале

deactivate
